# Positioning Error Analysis and Visualization

This repository contains two primary R scripts designed to analyze Global Navigation Satellite System (GNSS) positioning errors during ionospheric storm events (Storms 1тАУ4).

## 1. Scripts Overview

### `make_time_series.R`
**Purpose:** This is the primary data processing and temporal analysis script.
- **Data Parsing:** It scans directories for `.pos` files, extracts GPST (GPS Time) data, and aligns it with geographic coordinates.
- **Distance Calculation:** It calculates geodesic distances (using the `geosphere` library) between raw data points and a fixed reference coordinate (`ref_coords`) to determine:
    - Horizontal positioning errors (meters).
    - Absolute latitude/longitude deviations.
    - Vertical (height) errors.
- **Statistical Output:** - Generates summary statistics (Min, Q1, Median, Mean, Q3, Max, SD) grouped by hour.
    - Formats and prints LaTeX-ready tables for academic publication.
- **Visualizations:** Produces multi-day time series plots showing the fluctuation of horizontal errors across the duration of each storm event.
- **Output Files:** Saves processed data for each individual storm/day as a CSV file.

### `make_circular_plot.R`
**Purpose:** Generates advanced circular (polar) distribution plots for the manuscript.
- **Proportional Analysis:** It reads the CSV files generated by the time-series script and bins positioning errors into categorical ranges.
- **Circular Visualization:** - Uses `ggplot2` with `coord_polar` to display error distributions by hour of the day.
    - Features a "hollow center" design with a specialized "gap" at the 12 o'clock position for percentage scales.
- **Formatting:** Applies scientific notation (`plotmath`) to legend labels and ensures all labels are tangentially rotated for readability.
- **Output Files:** Saves high-resolution PDF files.

## 2. Requirements

### R Packages
Ensure you have the following libraries installed:
```r
install.packages(c("tidyverse", "dplyr", "tidyr", "lubridate", "geosphere", "ggplot2", "readr", "stringr", "extrafont"))
```

### System Dependencies
- **Fonts:** The scripts use "DejaVu Sans". You may need to install this font on your system and run `extrafont::font_import()`.
- **PDF Rendering:** The scripts use `cairo_pdf` for high-quality vector output, requiring a working Cairo installation on your OS.

---

## 4. Usage

1.  **Process Data:** Run `make_time_series.R` first. This will crawl the "Storm" folders, calculate errors, generate the necessary `.csv` files, and produce the time-series PDF plots.
2.  **Generate Circular Plots:** Run `make_circular_plot.R`. It will look for the `.csv` files produced in the previous step and generate the polar distribution plots.

---

## 5. Reference Coordinates
The scripts use the Sgonico, Trieste reference point for error calculation:
- **Longitude:** 13.763519885584342
- **Latitude:** 45.70975671789818
- **Height:** 323.4 m
